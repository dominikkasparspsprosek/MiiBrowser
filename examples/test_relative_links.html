Relative Links Test Page

Test Links:

  • Absolute path (starts with /)
  • Relative path (same directory)
  • Parent directory
  • Absolute URL (https)
  • Absolute URL (http)

Explanation:

The <base> tag in the document head tells the browser how to resolve relative URLs. Without it, relative paths like /search would be resolved as file:///C:/Users/.../search instead of https://example.com/search.

How to Test:

  1. Load this page from a real website (e.g., upload to a test server)
  2. Or use a Python HTTP server: python -m http.server 8000
  3. Click the relative links above
  4. They should resolve correctly based on the current page's URL
